مکمل Workflow — Step by Step

پہلی بار setup سے لے کر روزانہ استعمال تک

1
Pipeline کھولیں اور API Key ڈالیں
pipeline.html کھولیں۔ اوپر بائیں طرف اپنا AI provider چنیں (Claude، GPT، Gemini، Groq وغیرہ)۔ API key ڈالیں — یہ browser میں save ہو جائے گی۔
pipeline.html → AI Model Settings → Provider چنیں → API Key ڈالیں
2
Target Site کا URL ڈالیں
جس site سے خبریں لینی ہوں اس کا URL ڈالیں۔ Interval (30 منٹ)، Category، اور Rewrite Style چنیں۔ "Add Source" دبائیں۔
مثال: https://geo.tv/latest-news → Category: Pakistan → Style: Urdu → Add Source
3
Fetch کریں
"⟳ Fetch Now" دبائیں۔ Pipeline target site سے headlines اور content لے آئے گا اور Queue میں دکھائے گا۔ ہر 30 منٹ پر یہ خودکار ہو گا۔
Queue Tab → articles دکھیں گے → Status: NEW
4
AI Rewrite کریں
"✦ AI Rewrite All" دبائیں۔ AI ہر article کو نئے الفاظ میں لکھے گا — آپ کے chosen style میں (اردو، English، SEO، وغیرہ)۔ Status READY ہو جائے گی۔
AI Rewrite All → ہر article کی نئی headline، excerpt، اور مکمل مضمون
5
News Site پر Publish کریں
"↑ Publish All Ready" دبائیں۔ Articles shared localStorage میں چلے جاتے ہیں۔ News Site (index.html) کھولیں — اوپر notification آئے گا۔ "ابھی Import کریں" دبائیں۔
Publish → index.html → notification → Import کریں → Articles live!
6
روزانہ کا معمول
Pipeline میں sources ایک بار add کریں — پھر یہ خود کار ہے۔ ہر 30 منٹ پر fetch ہوگا۔ News Site ہر 30 سیکنڈ میں check کرتی ہے۔ آپ کو صرف articles review کر کے publish کرنا ہے۔
Setup once → Pipeline runs automatically → Site notifies → One click import
